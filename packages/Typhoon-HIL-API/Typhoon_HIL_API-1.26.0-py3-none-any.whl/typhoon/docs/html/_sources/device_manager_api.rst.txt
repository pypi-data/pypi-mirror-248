####################
Device Manager API
####################

**Module:** ``typhoon.api.device_manager``

Device Manager API set of functions/methods to manipulate current HIL setup
(add or remove devices), create new ones from scratch programmaticaly and
connect/disconnect current setup.
This is most commonly used for creating scripts for **testing** and
**automating** repetitive tasks, but its use is not restricted for these use
cases only.
