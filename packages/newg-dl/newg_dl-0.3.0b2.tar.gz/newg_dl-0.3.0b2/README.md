# Newgrounds Download

A Python library for downloading music from Newgrounds.

# Example Usage

Download:
```py
from newg-dl import ngdl

song_id = 1154735
audio_url = ngdl.download_music(song_id)
```

Get Data:
```py
from newg-dl import ngdl

song_id = 1154735
audio_url = ngdl.get_data(song_id)
```
